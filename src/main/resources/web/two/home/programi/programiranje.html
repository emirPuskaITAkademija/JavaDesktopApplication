Odsjek programiranja

Školovanjem na Programming odsjeku pripremićete se za rad na cjelokupnom procesu projektovanja, izrade i implementacije softvera, od početne analize, planiranja i dizajna, preko kodiranja i testiranja, do instalacije i održavanja softvera na mjestu njegovog korišćenja. Školovanje podrazumijeva rad na konkretnim projektima, sa ciljem samostalne izrade softverskih rješenja.

Program školovanja na Programming odsjeku obuhvata oblasti razvoja algoritama, teorije programskih jezika, širokog spektra alata, jezika i tehnologija:

  • Visual Studio
  • .NET
  • C
  • C#
  • PHP
  • Java
  • JavaScript
  • Python

Osim toga, polaznik se upoznaje sa algoritamskim modelovanjem, objektno orijentisanim principima, strukturama podataka, web programiranjem, cloud developmentom, bazama podataka, SQL programiranjem, dizajnom interfejsa aplikacija, testiranjem i održavanjem softvera, elementima sigurnosti i zaštite podataka i računarskih sistema.
